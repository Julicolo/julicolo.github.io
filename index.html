Pokedex

A Pokedex web app that fetches data from PokeApi.

In development mode it caches all api requests using Express, to reduce stress on the PokeApi.
It loads more Pokemon as you scroll down the list to reduce amount of Api calls made overall.


When clicked on a Pokemon it will display some information about the Pokemon, such as: it's Pokedex entry, it's stats, typing, etc.
Features a toggle to change the sprites to their shiny variant.
