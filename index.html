Pokedex

A Pokedex web app that fetches data from PokeApi.
In development mode it caches all api requests to reduce stress on the PokeApi.
It loads more Pokemon as you scroll down the list to reduce amount of Api calls made.

When clicked on a Pokemon it will display some information regarding the Pokemon on the right side.
Features a toggle to make the Pokemon sprites appear as 'shiny'.
Search function is still to be implemented.
